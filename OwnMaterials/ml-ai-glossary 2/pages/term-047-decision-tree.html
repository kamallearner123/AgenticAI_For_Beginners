Decision Tree

Tree of decisions based on feature splits.

Worked example

  1. Define input features and target/output.
  2. Choose a baseline model or metric.
  3. Fit on training data; evaluate on validation set.
  4. Inspect errors; iterate with regularization/tuning.

When to use

  • Prefer this concept/model when its assumptions roughly hold.
  • Validate with a baseline and compare using multiple metrics.

Why it matters

Intuition

Tree of decisions based on feature splits.

Concrete example

Example: Classify loan approval based on rules.

Code Snippet

Try this minimal example (python).

# Decision Tree for classification
from sklearn.datasets import load_wine
from sklearn.model_selection import train_test_split
from sklearn.tree import DecisionTreeClassifier
from sklearn.metrics import classification_report

X, y = load_wine(return_X_y=True)
Xtr, Xte, ytr, yte = train_test_split(X, y, test_size=0.25, random_state=42)

clf = DecisionTreeClassifier(max_depth=4, random_state=42).fit(Xtr, ytr)
print(classification_report(yte, clf.predict(Xte)))
