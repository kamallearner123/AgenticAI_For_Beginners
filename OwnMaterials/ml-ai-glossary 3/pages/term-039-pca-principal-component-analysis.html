PCA (Principal Component Analysis)

Linear technique to project data onto fewer orthogonal components.

Why it matters

PCA (Principal Component Analysis) is commonly used in the Dimensionality category. Understanding it helps you make better choices when designing pipelines, evaluating models, and communicating results.

Intuition

Linear technique to project data onto fewer orthogonal components.

Concrete example

Example: Compress 100D data to 2D for visualization.

Code Snippet

Try this minimal example (python).

# PCA to 2D with visualization (if matplotlib available)
from sklearn.datasets import load_digits
from sklearn.decomposition import PCA
from sklearn.preprocessing import StandardScaler
import matplotlib.pyplot as plt

X, y = load_digits(return_X_y=True)
X = StandardScaler().fit_transform(X)
X2 = PCA(n_components=2, random_state=0).fit_transform(X)

plt.figure()
plt.scatter(X2[:,0], X2[:,1], s=8, c=y)
plt.title("Digits PCA (2D)")
plt.show()
